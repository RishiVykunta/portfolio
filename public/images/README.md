# College Logo

Place your college logo image here.

**File name:** `iiitv-logo.png` (or update the path in `About.jsx`)

**Recommended specifications:**
- Format: PNG (with transparency) or SVG
- Size: 200x200px or similar square dimensions
- File size: Keep it under 500KB for faster loading

**To update the logo:**
1. Place your logo file in this `public/images/` folder
2. Name it `iiitv-logo.png` or update the `collegeLogo` variable in `src/components/About.jsx`
3. The component will automatically display the logo, or show a graduation cap icon if the logo is not found
